# 🚀 JST Clickzy

> A lightweight browser toolkit for productivity, PDF tools, screenshots, background removal, and AI shortcuts.

---

## ✨ About

JST Clickzy is a modern browser extension designed to make daily digital tasks easier and faster.

Instead of opening multiple websites and tools, JST Clickzy brings everything together in one clean and simple extension.

Built with:

* ⚡ Speed
* 🔒 Privacy
* 🎯 Simplicity
* 💻 Local Processing

---

# 🌟 Features

## 📄 PDF Tools

* Images → PDF
* PDF → Images

* Merge PDFs
* Split PDFs

# 🚀 Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Sbasnayaka/JST-Clickzy.git
```

---

## 2️⃣ Open Chrome Extensions

Go to:

```bash
chrome://extensions/
```

---

## 3️⃣ Enable Developer Mode

Turn on:
✅ Developer Mode

---

## 4️⃣ Load the Extension

Click:

```bash
Load unpacked
```

Then select the project folder.
